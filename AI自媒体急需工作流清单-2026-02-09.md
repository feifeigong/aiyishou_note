# AI自媒体急需工作流清单（2026-02-09）

## 按紧急度排序

1. 全网情报漏斗（先广后深）
- 目标：先抓新信号，再筛值得写的题。
- 组合：Google Alerts + OpenAlex API + Hugging Face Papers Trending + GitHub Watch（仅 Release）。
- 结果：每天稳定产出候选题池。

2. 深度核查工作流（防翻车）
- 目标：每个选题都能给出可追溯证据链。
- 组合：ChatGPT deep research / Claude Web Search + Citations。
- 结果：降低二手谣言复述风险。

3. NotebookLM 资料包工作流（专题化）
- 目标：同主题长期积累为可复用知识库。
- 组合：NotebookLM Discover Sources + 手动补充官方文档。
- 结果：专题写作速度持续提升。

4. 一稿多发改写流水线（效率核心）
- 目标：一份研究稿拆为多平台版本。
- 输出：公众号版 / 小红书版 / 短视频口播版 / 推文线程版。
- 结果：产能提升 2-4 倍，口径统一。

5. 自动化编排工作流（n8n/Make）
- 目标：抓取-筛选-写作-归档-复盘串联自动执行。
- 组合：n8n AI Agent 或 Make AI Agents。
- 结果：你专注判断与观点，重复劳动自动化。

## 先落地这三步

1. 情报漏斗 + 选题打分表（1天见效）
2. 深度核查模板（每篇必须留证据）
3. 一稿多发模板（显著压缩创作时间）
